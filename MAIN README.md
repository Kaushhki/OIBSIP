# Data Science Internship Projects

This repository contains all the tasks completed during my Data Science Internship at Oasis Infobyte. Each project focuses on applying machine learning and data analysis techniques to solve real-world problems.

---

#  Tasks Overview

#  Task 1: Iris Flower Classification
Built a classification model to predict iris flower species using multiple machine learning algorithms and compared their performance.

 

# Task 2: Unemployment Analysis with Python
Analyzed unemployment trends using data visualization techniques to understand the impact of different factors, especially during Covid-19.


#  Task 3: Car Price Prediction
Developed a regression model to predict car prices based on various features such as horsepower, mileage, and brand value.


#  Task 4: Email Spam Detection
Built a machine learning model using NLP techniques to classify emails as spam or not spam.



#  Task 5: Sales Prediction
Created a predictive model to estimate product sales based on advertising spend and other influencing factors.
